# CPGNet

Official repository for **CPGNet: Conditional Polarization Guidance for Camouflaged Object Detection**.

## Introduction
Camouflaged object detection (COD) aims to identify objects that are highly blended with their backgrounds.  
This repository contains the official implementation of **CPGNet**, a conditional polarization-guided framework for camouflaged object detection.

## Paper
- arXiv: https://arxiv.org/abs/2603.30008

## Highlights
- Conditional polarization guidance for RGB feature learning
- Progressive polarization-guided enhancement
- Edge-guided frequency refinement
- Designed for polarization-based camouflaged object detection
